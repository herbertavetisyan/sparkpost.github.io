Mini Hacks

Learn how to use SparkPost by completing these three challenges. Complete 2 mini hacks for a chance to win a prize. Complete the third for a second chance to win. Use the resources provided by the Developer Hub to help you along. Also, feel free to ask our onsite engineers for assistance.

Send An Email Using SparkPost

Using your new or existing SparkPost account, send an email to developers@sparkpost.com. Be sure to do the following:

  • Set the Subject to "SparkPost Lab: Mini Hack #1"
  • In the body of the email, include your first and last name and account email/username

Receive Data Via Webhooks

Let's demonstrate SparkPost's ability to push to a specified place. To complete this challenge, you must complete the following:

  • Set up a webhook that sends data to RequestBin
  • Send an email to developers@sparkpost.com
    • Subject: SparkPost Lab: Mini Hack #2
    • Content: your first and last name and account email/username
  • View your RequestBin, copy the output and email it to developers@sparkpost.com
    • Subject: SparkPost Lab: Mini Hack #2 Data
    • Content: your first and last name, account email/username, and the output from request bin

    Get Data Via Message Events

    SparkPost also gives you the ability to pull specific data about a transmission. Do the following to complete this challenge:

    • Query the API for message events where:
      • The from parameter is the datetime you performed the first mini-hack
      • The recipients parameter contains developers@sparkpost.com
    • Email the output to developers@sparkpost.com:
      • Subject: SparkPost Lab: Mini Hack #3
      • Content: your first and last name, account email/username, and the message events response
